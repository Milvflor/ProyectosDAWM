How to use .PEM (public key) file in Bitvise SSH Client & open server with Terminal

Bitvise SSH is a free SSH and SFTP client and server for Windows environments. As for any other type of remote connection, in order to establish secure connections we need both a server and a client.

Bitvise SSH Client is the client, which is used to establish remote connections to the SSH protocol. This client is used to establish a remote connection to the server terminal to control it, to configure and monitor port forwarding and to transfer files securely through Secure-FTP. This client is the one that must be installed on the machine from which we are going to connect to the server.

First, Open Bitvise SSH Client Software and click on Client key manager.

Import .pem file and if it have a passphrase introduce it.

It should be add on Client Key Manager as Global 1 location.

Then, click on Options Tab and check box Open in Terminal.

Now, click on Login tab, introduce (example):

Port: 22
Username: yourUsername
Initial method: publickey
Client Key: Global 1 (Key added in 4th step)
Passphrase: KeyPemFile

Then, click on Log in and Accepted for this Session.

Finally, the terminal should be open server successfully.
